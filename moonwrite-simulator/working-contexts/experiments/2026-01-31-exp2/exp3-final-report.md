# 후속 실험 리포트: 모델 사용 및 해상도 비교

> **실험일**: 2026-01-31
> **데이터**: 45개 샘플 슬라이드
> **주요 변경**: 모든 옵션에서 Pro 모델 사용, 해상도별 품질 비교

---

## 1. 실험 개요

### 1.1 실험 조건

**모델 사용 조건** (모든 옵션에서 Pro 사용):

| 옵션 | Flash 역할 | Pro 역할 |
|-----|----------|---------|
| Quick Draft | 추출 | **검증** |
| Standard | CoT 추출 | **분류 정제 + Gap 분석** |
| Premium | Self-Critique | **전체 검증** |

**의미 세분화 해상도**:

| 해상도 | 슬라이드당 유닛 | 용도 |
|--------|---------------|------|
| Low | 1-2개 | 개요/목차 |
| Medium | 1-3개 | 일반 권장 |
| High | 1-5개 | 정밀 검색 |

---

## 2. 옵션별 Pro 사용 효과

### 2.1 Quick Draft + Pro (Low Resolution)

| 지표 | 결과 |
|------|------|
| 총 유닛 | 52개 |
| 평균 유닛/슬라이드 | **1.16개** |
| Pro 수용 | 23.1% |
| **Pro 보완** | **63.5%** |
| Pro 거부 | 13.5% |

**핵심 발견**: Pro 검증으로 **63.5%의 유닛이 보완**됨. Quick Draft에서도 Pro 사용 시 품질 향상 효과 명확.

### 2.2 Standard + Pro (Medium Resolution)

| 지표 | 결과 |
|------|------|
| 총 유닛 | 89개 |
| 평균 유닛/슬라이드 | **1.98개** |
| **분류 정제** | **38.2%** |
| Gap 식별 | 4개 |
| Thesis 정렬 | 72% |

**핵심 발견**: Pro 분류 정제로 **38.2%의 카테고리가 수정**됨. Gap 분석 추가로 보완점 명확화.

### 2.3 Premium + Pro (High Resolution)

| 지표 | 결과 |
|------|------|
| 총 유닛 | 126개 |
| 평균 유닛/슬라이드 | **2.80개** |
| **Premium 품질** | **81.7%** |
| 이미지 기반 유닛 | 26.2% |
| 클러스터 수 | 13개 |

**핵심 발견**: Pro 전체 검증으로 **81.7%가 Premium 품질** 달성. 심층 멀티모달 분석 효과적.

### 2.4 옵션별 Pro 사용 효과 비교

| 옵션 | Pro 역할 | 품질 개선 효과 |
|-----|---------|--------------|
| Quick Draft | 검증 | 63.5% 보완 |
| Standard | 분류 정제 | 38.2% 카테고리 수정 |
| Premium | 전체 검증 | 81.7% Premium 품질 |

**결론**: **모든 옵션에서 Pro 사용 시 품질 향상** 확인. 특히 Quick Draft에서 효과가 가장 극명함.

## 3. 해상도별 품질 비교

### 3.1 정량적 비교

| 해상도 | 총 유닛 | 평균 유닛 | 정보 커버리지 |
|--------|--------|----------|--------------|
| **Low (1-2)** | 65 | 1.44 | Low (66.7% 이미지 누락) |
| **Medium (1-3)** | 92 | 2.04 | **Balanced** |
| **High (1-5)** | 139 | 3.09 | High (40% 소형 유닛) |

### 3.2 Low Resolution (1-2 유닛)

**장점**:
- 빠른 처리, 최소 비용
- 핵심 주제 빠른 파악
- 목차/개요 생성에 적합

**단점**:
- **66.7% 이미지 정보 누락** (심각)
- 세부 설명 35.6% 압축
- 논문 작성에 **부적합**

**적합 용도**: 개요/목차 생성만

### 3.3 Medium Resolution (1-3 유닛)

**장점**:
- 균형 잡힌 세분화 (56%가 2개 유닛)
- 논문 구조와 좋은 매핑
- 과도한 분할/압축 방지

**단점**:
- 일부 세부 기술 정보 손실 가능
- 복잡한 슬라이드에서 추가 유닛 필요

**적합 용도**: **일반 논문 작성 (권장)**

### 3.4 High Resolution (1-5 유닛)

**장점**:
- 높은 정보 커버리지 (텍스트 83.5%, 이미지 92.7%)
- 정밀한 검색 지원
- 다단계 추상화 가능

**단점**:
- **40% 소형 유닛** (과도한 세분화)
- 클러스터링 복잡도 높음 (119점)
- 처리 오버헤드 증가

**적합 용도**: 정밀 검색, 상세 방법론 추출

## 5. 핵심 발견

### 5.1 Pro 모델 사용 효과

1. **모든 옵션에서 Pro 사용 시 품질 향상**
   - Quick Draft: 63.5% 보완 (가장 큰 효과)
   - Standard: 38.2% 분류 정제
   - Premium: 81.7% Premium 품질

2. **Pro 역할 분화가 효과적**
   - 검증용 (Quick) vs 정제용 (Standard) vs 전체 검증 (Premium)

### 5.2 해상도 선택 가이드라인

1. **Low (1-2)는 개요용으로만 적합**
   - 66.7% 이미지 누락으로 논문 작성에 부적합

2. **Medium (1-3)이 일반 권장**
   - 균형 잡힌 커버리지
   - 논문 구조와 좋은 매핑

3. **High (1-5)는 주의해서 사용**
   - 40% 소형 유닛으로 과도한 세분화 위험
   - 정밀 검색/상세 방법론에만 적합

### 5.3 최적 조합 권장

| 시나리오 | 권장 조합 |
|---------|----------|
| 빠른 개요 확인 | Quick + Low |
| **일반 논문 작성** | **Standard + Medium** |
| 고품질 제출물 | Premium + High |
| 이미지 많은 연구 | Premium + Medium |

## 6. 결론

### 6.1 주요 결론

1. **Pro 모델은 모든 옵션에서 사용해야 함**
   - Quick Draft에서도 63.5% 품질 보완 효과

2. **Medium 해상도 (1-3)가 일반 권장**
   - Low는 이미지 누락 심각
   - High는 과도한 세분화 위험

3. **Standard + Medium + Pro 조합이 기본 권장**
   - 적정 비용 ($0.08)
   - 좋은 품질 (38.2% 정제, 72% Thesis 정렬)
