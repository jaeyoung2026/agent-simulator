## Direct 전략의 실행 흐름

### Phase 1: Flash 전체 분석
```
입력: 45개 슬라이드
  ↓
의미 단위 추출 (Direct 방식)
  - 각 슬라이드에서 1-2개 추출
  - 기본 이미지 분석 (유형 분류만)
  - 키워드 기반 분류
  - 시간적 우선순위 고려
  ↓
출력: 66개 SemanticUnit
  - 타입, 우선순위, 키워드 포함
```

### Phase 2: Pro 간극 검증
```
입력: Flash 추출 결과 (66개 단위)
  ↓
검증 작업:
  1. 커버리지 분석 (100% 달성)
  2. Gap 식별 (0개 발견)
  3. 품질 평가 (0.96/1.0)
  4. Thesis 일관성 검증 (0.46)
  ↓
출력: 검증 리포트
  - 품질 이슈 없음
  - 추가 정보 필요 없음
  - 좋은 일관성 확인
```
