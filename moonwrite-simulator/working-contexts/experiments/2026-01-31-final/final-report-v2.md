## 2. 핵심 변경: 모든 옵션에서 이미지 분석

### 2.1 옵션별 이미지 분석 전략

**모든 옵션에서 이미지 분석을 수행**하되, 분석 **깊이와 품질**에 차이를 둠:

| 분석 요소 | Quick Draft | Standard | Premium |
|----------|-------------|----------|---------|
| **유형 분류** | ✓ (기본) | ✓ | ✓ |
| **내용 설명** | 간략 (1문장) | 상세 (2-3문장) | 심층 (단락) |
| **연구 역할** | ✗ | ✓ | ✓ (정량적 분석 포함) |
| **텍스트 관계** | ✗ | ✗ | ✓ |
| **재현성 정보** | ✗ | ✗ | ✓ |
| **교차 이미지 분석** | ✗ | ✗ | ✓ |

### 2.2 이미지 분석 예시 비교

**동일 이미지 (학습 곡선 그래프)에 대한 분석:**

**Quick Draft**:
```
유형: graph
설명: 학습 그래프
```

**Standard**:
```
유형: training_curve
설명: 학습 초기 warm-up 구간에서 reward가 점진적으로 증가하다가 안정화되는 곡선
역할: 학습 안정성 확보 전략의 효과 입증
```

**Premium**:
```
유형: training_curve
시각적 요소: [학습 곡선, warm-up 구간 표시, 수렴 지점]
내용 설명: X축은 학습 스텝, Y축은 누적 보상. 학습 초기 warm-up 구간에서 reward가
          점진적으로 증가하다가 1M steps 이후 안정적으로 수렴하는 패턴 관찰.
텍스트 관계: "Reward 2, warm_up" 슬라이드 제목과 직접 연결
연구 기여도: 학습 안정성 확보 전략의 효과 입증. Ablation study에서 warm-up 유무 비교 가능.
정량적 인사이트: warm-up 구간 약 1M steps, 이후 안정적 수렴
재현성 정보: warm-up 스케줄 파라미터 명시 필요
```

## 4. 정성적 비교

### 4.1 Quick Draft

**특징**: 빠른 처리, 최소 비용, 기본 품질

**장점**:
- 가장 빠른 처리 속도 (키워드 기반 분류)
- 최소 비용 ($0.012)
- 전체 논문 구조를 빠르게 파악 가능
- **기본 이미지 분석으로 시각 자료 활용 가능**

**단점**:
- 분류 정확도가 Standard/Premium보다 낮음
- 이정표/글루 문장 없어 논문 흐름 단절
- Thesis 연결성 부족
- 이미지 분석이 간략하여 연구 맥락 파악 제한

**적합한 시나리오**:
- 빠르게 논문 구조 확인
- 초안 스케치
- 비용 제약이 있는 경우
- 시간이 촉박한 경우

### 4.2 Standard

**특징**: 균형 잡힌 품질, 적정 비용, 표준 기능

**장점**:
- 균형 잡힌 분류 분포
- CoT로 분류 근거 명확
- 이정표/글루 문장으로 논문 흐름 자연스러움
- **이미지 역할 분석으로 시각 자료의 연구 기여 파악**
- Gap 분석으로 보완점 식별
- 비용 대비 높은 품질

**단점**:
- Premium 대비 이미지 분석 깊이 제한
- Thesis 증거 연결 부족
- 역개요 검증 없음

**적합한 시나리오**:
- 일반적인 논문 초안 작성 (권장 기본 옵션)
- 학회 발표 준비
- 비용-품질 균형이 필요한 경우

### 4.3 Premium

**특징**: 최고 품질, 전체 기능, 높은 비용

**장점**:
- 상세한 Thesis (질문+주장+증거)
- thesisConnection으로 모든 섹션이 핵심 주장과 연결
- **심층 멀티모달 이미지 분석** (재현성 정보, 정량적 인사이트)
- 역개요 검증으로 Thesis 정렬도 확인 (89%)
- 구체적 플레이스홀더 (실험 제안 포함)
- Gap 분석의 심각도/우선순위 포함
- 미정렬 유닛 식별 및 제외 권장

**단점**:
- 처리 시간 가장 길음
- 비용 가장 높음 ($0.160, 13.3x)
- 소규모 프로젝트에는 과도할 수 있음

**적합한 시나리오**:
- 중요한 학회/저널 제출
- 최종 논문 품질이 중요한 경우
- 이미지/그래프가 많은 연구
- 시간 여유가 있는 경우

## 5. 핵심 발견

### 5.1 이미지 분석의 가치

**모든 옵션에서 이미지 분석을 수행**한 결과:

| 옵션 | 이미지 활용 효과 |
|-----|-----------------|
| Quick Draft | 기본 유형 분류로 결과/방법 구분 보조 |
| Standard | 이미지 역할 분석으로 분류 정확도 향상, Gap 발견 보조 |
| Premium | **이미지 기반 정량적 인사이트 추출**, 재현성 정보 제공 |

**결론**: 연구 슬라이드에서 이미지 분석은 **모든 품질 수준에서 유용**하며, 분석 깊이에 따라 기여도가 달라짐

### 5.2 옵션 차별화의 본질

**핵심 인사이트**: 옵션 간 차이는 **특정 과업의 수행 유무가 아닌, 동일 과업의 품질/깊이 차이**

| 과업 | Quick Draft | Standard | Premium |
|-----|-------------|----------|---------|
| 의미 추출 | 기본 추출 | CoT 추출 | Self-Critique 추출 |
| 이미지 분석 | 유형 분류 | 유형+역할 | 심층 분석 |
| 클러스터링 | 키워드 기반 | 하이브리드 | Thesis-First |
| Writing Principles | 최소 적용 | 표준 적용 | 전체 적용 |
| Gap 분석 | 없음 | 기본 분석 | 상세 분석 |

### 5.3 비용-품질 트레이드오프

```
품질 ↑
│                    ★ Premium ($0.16)
│
│           ★ Standard ($0.047)
│
│  ★ Quick Draft ($0.012)
│
└──────────────────────────────→ 비용 ↑
```

**권장 선택 기준**:
- **$0.02 미만 예산**: Quick Draft
- **$0.05 이하 예산 + 좋은 품질**: Standard (권장)
- **품질 최우선**: Premium

## 6. 최종 권장 사항

### 6.1 옵션별 추천

| 시나리오 | 권장 옵션 | 이유 |
|---------|----------|------|
| 빠른 구조 확인 | Quick Draft | 최소 비용, 1분 내 처리 |
| **일반 논문 작성** | **Standard** | **균형 잡힌 품질, 적정 비용** |
| 중요 제출물 | Premium | 최고 품질, 역개요 검증 |
| 이미지 많은 슬라이드 | Premium | 심층 멀티모달 분석 |
| 비용 제약 있는 경우 | Quick Draft → 필요시 Standard | 점진적 업그레이드 |

### 6.2 기본 옵션 권장

> **Standard 옵션 (CoT + 하이브리드 + 표준 Writing Principles + 표준 이미지 분석)이 일반 사용에 권장된다.**
>
> - Quick Draft 대비: 분류 균형 + Thesis 연결 + 이정표/글루 문장 + 이미지 역할 분석
> - Premium 대비: 72% 비용 절감 + 충분한 품질

### 6.3 가격 책정 제안

| 옵션 | 상대 비용 | 가격 예시 | 비고 |
|-----|----------|----------|------|
| Quick Draft | 1x | 무료 or 기본 | 체험/빠른 확인용 |
| Standard | 4x | 중간 요금 | 기본 권장 |
| Premium | 13x | 프리미엄 요금 | 중요 문서용 |

### 6.4 구현 우선순위

```
1단계: Standard 옵션 구현 (기본값)
  - CoT 추출
  - 표준 이미지 분석 (유형+역할)
  - 하이브리드 분류
  - 이정표/글루 문장/Gap 분석

2단계: Quick Draft 옵션 추가 (저비용)
  - Direct 추출
  - 기본 이미지 분석 (유형)
  - 키워드 분류
  - 최소 Writing Principles

3단계: Premium 옵션 추가 (고품질)
  - Self-Critique + 심층 이미지 분석
  - Thesis-First 4단계 클러스터링
  - 전체 Writing Principles
  - 역개요 검증
```

## 8. 결론

### 8.1 주요 개선 사항

1. **모든 옵션에서 이미지 분석 수행**: Quick Draft도 기본 이미지 분석 포함
2. **옵션 차별화 명확화**: 과업 유무가 아닌 품질/비용/속도 차이로 구분
3. **샘플 확대**: 20개 → 45개로 결과 신뢰성 향상
4. **Writing Principles 수준별 적용**: 모든 옵션에서 수준에 맞게 적용

### 8.2 핵심 결론

| 핵심 발견 | 결론 |
|----------|------|
| 이미지 분석 | 모든 옵션에서 수행해야 함, 깊이로 차별화 |
| 옵션 차이 | 품질/비용/속도 트레이드오프로 명확히 구분 |
| 기본 옵션 | Standard 권장 (품질-비용 균형) |
| 고품질 필요시 | Premium 선택 (13배 비용, 최고 품질) |
